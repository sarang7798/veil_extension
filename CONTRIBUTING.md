# Contributing to Veil

Thank you for your interest in Veil.

## No public merges

This repository **does not accept pull requests or merges from the public**.

- **Do not open PRs** expecting them to be reviewed or merged here.
- Pull requests opened against this repository will be **closed without merge**.
- There are no outside collaborators with write access, and public contributions are not merged into this codebase.

## Fork for personal experiments

You are welcome to **fork** the repository for personal, non-commercial experiments under the terms of the [LICENSE](LICENSE).

Forks are for your own use and learning. Please do not assume that changes made in a fork will be pulled upstream.

## Collaboration

If you want to collaborate (features, partnerships, licensing, or commercial use), contact the maintainer directly:

- **Maintainer:** [@sarang7798](https://github.com/sarang7798)

Do not use a public pull request as the primary channel for collaboration requests.
